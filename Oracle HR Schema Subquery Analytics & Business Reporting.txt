Oracle HR Schema Subquery Analytics & Business Reporting

 1. Salary Analysis Using Subqueries
	a. Employees Earning More Than the Company Average Salary

		Subquery Type: Scalar Subquery
		-----
		SELECT Employee_ID, First_Name, Last_Name, Salary
		FROM Employees
		WHERE Salary >(SELECT AVG(Salary)
  	    	 FROM Employees);
--

Logic: The subquery calculates the company average salary. The outer query returns employees whose salary exceeds that value.

------------
b. Employees Earning Less Than the Company's Highest Salary

	Subquery Type: Scalar Subquery
	------
	SELECT Employee_ID, First_Name, Last_Name, Salary
	FROM Employees
	WHERE Salary < (SELECT MAX(Salary)
	);
--

Logic: The highest salary is retrieved first, then employees earning less than that amount are displayed.

-------------

c. Departments Whose Average Salary Exceeds the Company Average Salary

	Subquery Type: Scalar Subquery with GROUP BY

	SELECT Department_ID
	FROM Employees
	GROUP BY Department_ID
	HAVING AVG(Salary) > (SELECT AVG(Salary)
        FROM Employees
	);
--
Logic: Compares each department's average salary with the overall company average.

------------

d. Employees Earning the Highest Salary Within Their Department

	Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Last_Name, Salary, Department_ID
	FROM Employees E
	WHERE Salary =(SELECT MAX(Salary)
        FROM Employees
        WHERE Department_ID = E.Department_ID
      	);
--

Logic: The subquery is executed for each employee's department to find the highest salary in that department.

2. Department & Location Analysis

a. Employees Working in Departments Located in the United States

	Subquery Type: Nested Subquery

	SELECT Employee_ID, First_Name, Last_Name
	FROM Employees
	WHERE Department_ID IN (
	SELECT Department_ID
    	FROM Departments
    	WHERE Location_ID IN
    	(
        SELECT Location_ID
        FROM Locations
        WHERE Country_ID = 'US'
    	)
	);

Logic: Finds United States locations, then departments in those locations, then employees in those departments.

-----------------

b. Departments That Currently Contain Employees

	Subquery Type: Multi-Row Subquery


	SELECT Department_ID, Department_Name
	FROM Departments
	WHERE Department_ID IN 
	(
   	SELECT Department_ID
    	FROM Employees
    	WHERE Department_ID IS NOT NULL
	);
--

Logic: Returns departments referenced by at least one employee.

------------------

c. Departments That Currently Have No Employees

	Subquery Type: Multi-Row Subquery

	SELECT Department_ID, Department_Name
	FROM Departments
	WHERE Department_ID NOT IN
	(
    	SELECT Department_ID
    	FROM Employees
    	WHERE Department_ID IS NOT NULL
	);
--

Logic: Returns departments not referenced by any employee.

-----------------

d. Employees working in departments located in the Europe region using nested subqueries.

	Subquery Type: Nested Subquery

	SELECT Employee_ID, First_Name, Last_Name
	FROM Employees
	WHERE Department_ID IN
	(
    	SELECT Department_ID
    	FROM Departments
    	WHERE Location_ID IN
    	(
        SELECT Location_ID
        FROM Locations
        WHERE Country_ID IN
        (
        SELECT Country_ID
        FROM Countries
        WHERE Region_ID =
        (
        SELECT Region_ID
        FROM Regions
        WHERE Region_Name = 'Europe'
        )
        )
    	)
	);
--

Logic: Filters employees through department, location, country, and region hierarchy.

 3. Manager & Hierarchy Analysis

a. Employees Earning More Than Their Manager

	Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Salary
	FROM Employees E
	WHERE Salary >
      	(
        SELECT Salary
        FROM Employees M
        WHERE M.Employee_ID = E.Manager_ID
      	);
--

Logic: Compares each employee's salary against their manager's salary.

-----------------------------

b. Managers supervising at least one employee earning more than 10,000.

	Subquery Type: Multi-Row Subquery

	SELECT Employee_ID, First_Name, Last_Name
	FROM Employees
	WHERE Employee_ID IN
	(
    	SELECT Manager_ID
    	FROM Employees
    	WHERE Salary > 10000
	);
--

Logic: Returns managers who supervise at least one employee earning above 10,000.

-----------------

c. Employees Whose Manager Works in the Same Department

Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Department_ID
	FROM Employees E
	WHERE Department_ID =
      	(
        SELECT Department_ID
        FROM Employees M
        WHERE M.Employee_ID = E.Manager_ID
      	);

------------------

d. Employees Whose Manager Works in a Different Department

	Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Department_ID
	FROM Employees E
	WHERE Department_ID <>
      	(
        SELECT Department_ID
        FROM Employees M
        WHERE M.Employee_ID = E.Manager_ID
      	);
------------------

4. Inline View & Query Execution

a. Filtering employees earning more than 10,000.

	Subquery Type: Inline View
	
	SELECT *
	FROM
	(
    	SELECT Employee_ID, First_Name, Salary
    	FROM Employees
	)
	WHERE Salary > 10000;

---

b. Displaying departments with average salary greater than 9,000.

	Subquery Type: Inline View

	SELECT *
	FROM
	(
    	SELECT Department_ID, AVG(Salary) AS Avg_Salary
    	FROM Employees
    	GROUP BY Department_ID
	)
	WHERE Avg_Salary > 9000;
---------------

c. Displaying departments with more than five employees.

	Subquery Type: Inline View

	SELECT *
	FROM
	(
    	SELECT Department_ID, COUNT(*) AS Employee_Count
    	FROM Employees
    	GROUP BY Department_ID
	)
	WHERE Employee_Count > 5;
------------------

5. Advanced Subquery Logic
a. Employees earning above their department average salary.

	Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Salary
	FROM Employees E
	WHERE Salary >
      	(
        SELECT AVG(Salary)
        FROM Employees
        WHERE Department_ID = E.Department_ID
      	);
------------------

b. Employees earning above the average salary for their job role.

	Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Salary, Job_ID
	FROM Employees E
	WHERE Salary >
      	(
        SELECT AVG(Salary)
        FROM Employees
        WHERE Job_ID = E.Job_ID
      	);
-------------------

c. Departments where all employees earn more than 5,000.

	Subquery Type: Subquery with HAVING

	SELECT Department_ID
	FROM Employees
	GROUP BY Department_ID
	HAVING MIN(Salary) > 5000;

Logic: If the minimum salary is above 5000, then all employees in that department earn more than 5000.

------------------

d. Employees who are the only employee in their department.

	Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Department_ID
	FROM Employees E
	WHERE 1 =
      	(
        SELECT COUNT(*)
        FROM Employees
        WHERE Department_ID = E.Department_ID
      	);

6. Executive HR Analytics Challenge
a. Employees earning above the company average salary.

	Subquery Type: Scalar Subquery

	SELECT Employee_ID, First_Name, Salary
	FROM Employees
	WHERE Salary >
      	(
        SELECT AVG(Salary)
        FROM Employees
      	);
---

b. Departments with more employees than Department 50.

	Subquery Type: Scalar Subquery

	SELECT Department_ID, COUNT(*) AS Employee_Count
	FROM Employees
	GROUP BY Department_ID
	HAVING COUNT(*) >
      	(
        SELECT COUNT(*)
        FROM Employees
        WHERE Department_ID = 50
      	);
------

c. Employees working in the department with the highest average salary.

	Subquery Type: Nested Subquery

	SELECT Employee_ID, First_Name, Department_ID
	FROM Employees
	WHERE Department_ID =
	(
    	SELECT Department_ID
    	FROM
    	(
        SELECT Department_ID,
        AVG(Salary) Avg_Salary
        FROM Employees
        GROUP BY Department_ID
        ORDER BY Avg_Salary DESC
    	)
    	WHERE ROWNUM = 1
	);
--------

d. Employees earning the second-highest salary.

	Subquery Type: Nested Subquery

	SELECT Employee_ID, First_Name, Salary
	FROM Employees
	WHERE Salary =
	(
    	SELECT MAX(Salary)
    	FROM Employees
    	WHERE Salary <
        (
        SELECT MAX(Salary)
        FROM Employees
        )
	);

---

e. Managers whose employees' average salary exceeds 10,000.

	Subquery Type: Correlated Subquery

	SELECT Employee_ID, First_Name, Last_Name
	FROM Employees M
	WHERE 10000 <
      	(
        SELECT AVG(Salary)
        FROM Employees E
        WHERE E.Manager_ID = M.Employee_ID
      	);
--
Logic: Calculates the average salary of employees under each manager and returns managers whose team average exceeds 10,000.
